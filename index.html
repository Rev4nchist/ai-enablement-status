New Project

Fourth Marketing Brain

An MCP server that connects Claude, ChatGPT, and Microsoft Copilot directly to curated marketing content on SharePoint. The marketing team's AI-powered knowledge base.

search_knowledge

Full-text search across curated marketing content. Find brand guidelines, campaign materials, and strategic docs instantly.

browse_library

Navigate the folder structure of marketing materials. Explore content categories and discover available resources.

ask_question

RAG-style Q&A with confidence levels — GROUNDED, PARTIAL, or NO ANSWER. AI knows when it doesn't know.

Capabilities

  • search_knowledge — full-text search
  • browse_library — folder navigation
  • get_document — retrieve document text
  • ask_question — RAG Q&A with confidence
  • list_content_areas — content categories

Architecture

  • Python MCP Server
  • SharePoint + Azure AD OAuth
  • Microsoft Graph API backend
  • Claude, ChatGPT, Copilot compatible
  • Local, HTTP, or Railway deploy
